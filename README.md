# Neural Networks Project 🧠
Proyecto de redes neuronales en Python con estructura moderna basada en src/ y gestionado con uv.

## Requisitos
- uv

## Instalación de uv (Windows)
winget install --id=astral-sh.uv

## Instalación del proyecto
- uv venv
- uv pip install -e .

## Ejecución
- uv run start
- uv run task lint
- uv run task format

## Resumen
- Código en src/
- uv pip install -e .
- uv run app
- uv run python src/main.py
- winget install --id=astral-sh.uv
