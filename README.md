# Neural Networks Project 🧠
Proyecto de redes neuronales en Python con estructura moderna basada en src/ y gestionado con uv.

## Instalación de uv
- (Windows) winget install --id=astral-sh.uv
- (mac) brew install uv

## Instalación del proyecto
- uv venv 
- uv pip install -e .

## Ejecución
- uv run task start
- uv run task lint
- uv run task format
